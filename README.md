REAL TIME CHAT APPLICATION

INTRODUCTION
This is a simple real time chat application where users can send and receive messages instantly. It is built using standard web technologies like HTML for structure, CSS for styling, and JavaScript for functionality.

HOW IT WORKS
The application uses a server to connect multiple users together. When one user sends a message, the server immediately shares it with everyone else connected to the chat.

HOW TO RUN THE PROJECT
1. Open the project folder in your code editor.
2. Open the terminal.
3. Type the command: node server.js
4. Once the server is running, open your web browser.
5. Go to the address: localhost:3300 to start chatting.

FEATURES
User friendly design with a dark theme.
Instant messaging without page reloads.
Works on all modern web browsers.
